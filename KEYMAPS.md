# Keymaps

List of custom and/or useful keymaps.

### workspaces and windows
- `Super + Number` - switch to `Number` workspace
- `Super + Shift + q` - close window
- `Super + f` - toggle focus window / full screen
- `Super + h/j/k/l` - change current active window

### search bar
- `Alt + Space` - open search bar
- `Super + c` - open clipboard history

### lock scrren, suspend, shutdown
- `Super + x` - lock screen
- `Ctl + Alt + Del` - suspend
- `Ctl + Alt + End` - shutdown

### display configs
- `Super + D` - switch between saved display configs
- `autorandr --save <name>` - save current display config

### reload configs
- `Super + Shift + R` - reload i3 config

### terminal
- `Alt + f` - fuzzy find and open project
- `Alt + j/k/l/ñ` - switch to different terminla inside same project
- `(Ctl + a) + (F) + (a-z)` - copy highlighted link
- `(Ctl + a) + (F) + (Ctl + a-z)` - open highlighted link

### mardown files
- `:PeekOpen` - open preview of markdown file inside neovim
- `:PeekClose` - close preview of markdown file inside neovim

### neovim
- `Space + gd` - go to definition
- `Ctl + o` - go to previous file
- `Space + d` - show diagnostics of current line
- `Ctl + n` - next option from predictions
- `Ctl + p` - previous option from predictions
